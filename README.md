# NAND2Tetris
NAND2Tetris is a course that helped me to understand the fundamental principles on how computers actually work by explaining the connections between the hardware and software platforms. The course begins with using NAND gates to create other gates/functions and ends with building a general-purpose computer on which a game can be played.

This course explains how the components in hardware, architecture, operating systems, programming languages, compilers, data structures and algorithms, and software engineering are used to build a fully-functioning computer system as taught in computer science courses with a hands-on approach.

The first four projects that construct the computer platform are built using a simple Hardware Description Language (HDL) and simulated on a hardware simulator. The five subsequent software projects can be written in any modern programming language, but I chose to use {}. The remaining three projects are written in the assembly language and high-level language implemented in previous projects.
